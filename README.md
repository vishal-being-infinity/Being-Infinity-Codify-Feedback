# Being Infinity Codify Feedback

This repository collects user feedback, bug reports, and feature requests submitted through the **Being Infinity Codify** website.  

All submissions are automatically sent here as GitHub Issues to help the team track and resolve them efficiently.

---

## How Feedback Works

Users can submit feedback via the website using a visual feedback button. Feedback is categorized as:

- **Bug** – Report any issues or unexpected behavior  
- **Feature** – Suggest new features or improvements  
- **Other** – General feedback or comments  

Each submission creates a GitHub Issue including:

- **Category** (Bug / Feature / Other)  
- **Title** (Short summary)  
- **Description** (Detailed explanation)  
- **Contact Email** (Optional, for follow-up)

---

## Guidelines

- Please check existing issues before adding manual entries.  
- Keep sensitive information out of issue descriptions.  
- This repository is **public**, but user emails are only used for follow-up if provided.
